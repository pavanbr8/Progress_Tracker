# Progress_Tracker
Summary: Provides an overview of total goals, completed milestones, and overall progress status.

Progress Dashboard: A visual dashboard featuring charts and KPIs to track daily, weekly, and overall performance.

Goals Data: Contains detailed records of learning objectives, topics, and completion status.

Excel Syllabus Data: Includes syllabus breakdown and topic-wise coverage to measure progress systematically.

Study Hours Data: Logs daily study hours, enabling time tracking and productivity analysis.
